# ant_raiding_patterns
A NetLogo simulation of ant raiding patterns with emergent characteristics

# To Run:

This simulation runs in NetLogo. You can download the NetLogo client from https://www.netlogo.org/ 

Once installed, open the client. Then:

1. download `ant_raiding_patterns.nlogox` from this repo
2. File > Open > `ant_raiding_patterns.nlogox`

and that's it...

Controls for the simulator can be found under the 'Info' tab in the NetLogo client.
